# KUEPA

### 1. Instalar paquetes laravel despues de clonar
```
composer install
```

### 2. Creacion de base de datos
```
CREATE DATABASE kuepa;
```

### 3. Ejecutar migraciones
```
php artisan migrate
```

### 4. Ejecutar seeders
```
php artisan db:seed
```

### 5. Iniciar servidor
```
php artisan serve
```
